title: 我的第一篇博文
date: 2016-02-03 10:39:49
tags:[“学习”]
categories:学习

最近写博客热，本人也不能免俗，抽空用hexo搭了一套静态博客，用于整理自己的知识库和心理路程。

因为工作关系，平时接触到技术非常杂，覆盖了很多前端后端的内容和项目，先不管静不精通，熟不熟练，凡是混个脸熟的先列一下清单：

  1. jsp + html
  2. js
  3. java
  4. oracle
  5. hadoop
  6. spring
  7. scala
  8. hive
  9. shell
  10. tomcat
  11. sqoop
  12. flume

以上就是我工作四年来的收获，虽然看着貌似不少，但是确实门门不精，块块有缺陷，仅仅达到能用的地步。

曾经我沉浸在自己的小世界里，无忧无虑的工作着，想学了学学，不想学就吃老本。买来书就放书柜里，下好教程就堆桌面，以为拿来了就是自己的了，结果……

还有几天就过年了，掐指一算，明年已然27了。。。马上要奔三的年纪了，还是这般晃荡，惭愧啊。

不得不再次痛下决心，立帖为证，充实自己，提高自己！

最后分享三句从别人那里抄来的话，警醒自己：

除了编程技巧和程序设计能力，解决问题的稳准狠是衡量一个程序员是否优秀的重要因素之一，也是资深技术人员真正的价值所在。
程序员真正的价值是什么？以最小的代价解决问题！知行合一，方可无敌于天下。
我不是懂得多，我只是学的快而已。
